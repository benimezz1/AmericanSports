# PlayNorth (protótipo)

Protótipo de portal de notícias de ligas americanas (NFL, NBA, NHL, MLB, MLS) em HTML/CSS/JS.

## Como rodar local
- Abra `index.html` em um servidor simples (recomendado), por exemplo:
  - VS Code: extensão "Live Server"
  - ou qualquer servidor estático

## Dados (teste)
- `data/news.json` (notícias)
- `data/games.json` (jogos)
- `data/standings.json` (rankings)
